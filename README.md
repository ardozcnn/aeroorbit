# AeroOrbit: 2D Uçuş Yörüngesi Simülatörü

Hava sürtünmesi, kütle ve yerçekimini diferansiyel denklemlerle hesaba katarak bir hava aracının uçuş yörüngesini hesaplayan ve görselleştiren fizik tabanlı Python simülasyonu.

# Özellikler

* SciPy kütüphanesi ile diferansiyel denklem çözümü
* Dinamik aerodinamik sürüklenme (drag) hesaplaması
* Matplotlib ile 2D yörünge görselleştirmesi

# Kurulum

Projeyi bilgisayarınızda çalıştırmak için öncelikle gerekli kütüphaneleri yükleyin:

pip install numpy matplotlib scipy
